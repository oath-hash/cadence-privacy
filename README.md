# cadence-privacy

Privacy policy for the Cadence iOS app, served via GitHub Pages.

The published URL is <https://oath-hash.github.io/cadence-privacy/> and is the privacy-policy link used in the Cadence App Store listing.

## Contents

- `index.html`: the privacy policy. Static HTML, system fonts, no JavaScript, no external resources.
- `LICENSE`: Creative Commons CC0 1.0, placing the policy text in the public domain.

## Updating

1. Edit `index.html` and update the effective date.
2. Commit and push to `main`. GitHub Pages rebuilds within a minute or two.
3. If a change affects what data is handled, note it in the Cadence App Store release notes for the version that ships alongside.
